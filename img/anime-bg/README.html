当前背景图片

本目录包含博客的动漫风格背景图片：

  • bg.jpg - 主背景图片（默认使用）
  • bg-sakura.jpg - 樱花风格背景
  • bg-night.jpg - 夜景风格背景

如何替换背景图片

方法1：替换本地图片

  1. 准备你喜欢的动漫风格图片（建议尺寸：1920x1080 或更高）
  2. 将图片重命名为 bg.jpg（或其他名称）
  3. 复制到此目录 D:\Blogfile\source\img\anime-bg\
  4. 如果使用其他文件名，需要修改 CSS 文件中的路径
